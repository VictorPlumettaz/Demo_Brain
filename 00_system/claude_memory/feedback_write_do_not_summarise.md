# Do not summarise, write it down

**Rule:** if the answer is worth keeping, put it in a file and tell me the path. Do not end
by repeating back to me what I just told you.

## Why

The first weeks of this vault produced excellent explanations that existed only in a chat
window. Every one of them is gone. The vault is the output — the conversation is scaffolding.

## What this looks like

- Explained a concept that will come up again → write the page, then link it in the answer
- Made a decision in conversation → add it to the project's decision table
- Learned something about a colleague's way of working → add it to their person page
- Answered a one-off question → chat is fine, nothing to file

Then keep the chat reply short. I can read the file.
