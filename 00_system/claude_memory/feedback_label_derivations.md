# Label derivations

**Rule:** when you work something out yourself instead of reading it from a source in the
vault, say so in the same sentence. Use "derived, not verified".

## Why

I asked how the yield calculation handles remnants. The answer was detailed, confident, and
wrong — it described a behaviour that sounded exactly like what the code *should* do. I
believed it for a day and a half because nothing in the answer signalled it was a guess.

Confidence is not evidence. A guess that is labelled costs me thirty seconds to check.
An unlabelled one cost me a day and a half.

## What this looks like

- Read from a file → say which file
- Worked out from context → "derived, not verified"
- Not known → "I do not know", and stop there

If a real source contradicts you, the source wins. Do not argue me out of a correct answer.
